五笔字根图片使用说明
====================

请将从键盘图上切下来的字根图片放在这个文件夹中。

图片命名规则：
- 格式: 键位_序号.png
- 键位: 大写字母 (A-Z，不包括Z)
- 序号: 从1开始的数字

示例：
- G_1.png  -> G键的第1个字根（如：王）
- G_2.png  -> G键的第2个字根（如：一）
- G_3.png  -> G键的第3个字根（如：五）
- F_1.png  -> F键的第1个字根（如：土）
- F_2.png  -> F键的第2个字根（如：士）
...以此类推
